# 🔌 Cabling Troubleshooting

## 🧩 Issue 1: No Link Light

### 🔍 Symptoms

* No LED activity on switch port
* Device not detected


### 🧠 Root Cause

* Faulty cable or incorrect crimping


### ✅ Resolution

* Re-crimped cable using T568B standard
* Tested with known working device


---

## 🧩 Issue 2: Intermittent Connectivity

### 🔍 Symptoms

* Packet loss
* High latency spikes


### 🧠 Root Cause

* Damaged cable or poor termination


### ✅ Resolution

* Replaced cable
* Verified using: ping 192.168.10.1


---

## 📚 Key Learning

* Always start with Layer 1
* Physical faults can mimic network issues
